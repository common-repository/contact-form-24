Simple and intuitive contact form plugin for Wordpress.

== Description ==

Contact form 24 made using the technology of Ajax. Very easy to use.  You activate the plugin and it works! Now on your screen there is a tab - Contact!

== Installation ==

1. Upload the entire `Contact Form 24` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

 == Frequently Asked Questions ==

= What settings in the plugin? =

In this version of the plugin you can set any colors. This way you can easily adjust the color gamut for your website. You activate the plugin through the 'Plugins' menu in WordPress, the plugin settings indicate your e-mail and you're done!
